# ADR 2: Biblioteca de Estilos y Enrutamiento Frontend (Bootstrap & React Router)

**Estado:** Aceptado  
**Contexto:** Necesitábamos una interfaz profesional y navegación fluida entre Login y Registro sin recargas de página (SPA).

**Decisión:** Uso de **Bootstrap 5** y **React Router Dom v6**.

**Consecuencias:** * **Positivas:** Maquetación rápida con componentes pre-diseñados y experiencia de usuario moderna.  
* **Negativas:** Carga mínima de CSS extra y necesidad de configurar rutas en el cliente.
