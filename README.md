# devops_uc3m

Repositorio de prueba con una calculadora simple desarrollada en React.js

## Calculadora Simple

Una calculadora funcional desarrollada con React.js y desplegada en GitHub Pages.

### Características

- ✅ Operaciones básicas: suma, resta, multiplicación, división
- ✅ Botón de limpiar (C)
- ✅ Cambio de signo (±)
- ✅ Porcentaje (%)
- ✅ Números decimales
- ✅ Diseño moderno inspirado en iOS
- ✅ Responsive

### Desarrollo Local

Para ejecutar la calculadora localmente:

```bash
cd calculator-app
npm install
npm start
```

La aplicación se abrirá en [http://localhost:3000/devops_uc3m](http://localhost:3000/devops_uc3m)

### Producción

Para construir la versión de producción:

```bash
cd calculator-app
npm run build
```

### Despliegue

La calculadora se despliega automáticamente en GitHub Pages cuando se hace push a la rama `main`.

URL de la calculadora: [https://rotlerxd.github.io/devops_uc3m/](https://rotlerxd.github.io/devops_uc3m/)
